# 个人简历模板

[https://zelda.github.io/Resume](https://zelda.github.io/Resume)

## 介绍

参考众多简历风格，@DIYgod，@ekCit以及我自己的博客主题 [itsay.tech](http://itsay.tech)

## 面向对象

前端求职/程序员求职

## Usage

1. 先Star本项目，然后Clone或者直接下载到本地
2. 修改index.html内相关信息
3. 微调样式（作为前端求职，这点应该不成问题）
4. 生成pdf(开发中)
5. 部署到线上
6. 生成访问二维码（开发中）
7. 祝您求职成功！
